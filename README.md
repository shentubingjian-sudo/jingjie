# 镜界 · Mirror Realm

> 沉浸式角色扮演剧情引擎 v2

镜界不是一个聊天机器人——它是一个由**规则驱动导演系统**调度体验的完整叙事引擎。

## 核心能力

- 🎬 **规则驱动导演系统**：6个导演动作，不看单轮数值看趋势，只在心流偏离时干预
- 🧠 **三层持久记忆**：工作记忆 + 情节记忆 + 持久记忆池，同池混存，语义命中
- 🃏 **剧情资产系统**：语义卡片 + 契诃夫之枪回收，作者写卡片，系统选时机
- 📊 **协议化双层输出**：用户看到叙事，系统读取结构化导演数值
- 🌱 **角色记忆成长**：游玩中自动塑造角色，一条条记忆条目自动生成并参与检索
- 💭 **Plot-based Reflection**：NPC 每5轮反思玩家行为，展现更深共情
- ⏳ **时间连续性**：离线世界仍在呼吸，NPC 会执行你的离线指令
- 📦 **角色可带走**：导出 JSON，跨场景、跨世界观迁移

## 安装

在 WorkBuddy 中说：

```
帮我安装 shentubingjian-sudo/jingjie
```

或命令行：

```bash
npx skills add shentubingjian-sudo/jingjie -g -y
```

## 使用

在 WorkBuddy 中说「开镜界」即可启动。支持单作品模式、世界观融合模式。
