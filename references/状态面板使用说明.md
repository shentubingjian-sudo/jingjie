# 镜界沉浸式状态面板使用说明

## 文件位置
`G:\cloud\2026-05-10-task-1\index.html`

直接在浏览器打开即可。这是一个纯前端页面，无需服务器。

## 面板功能

该网页是一个「世界状态辅助面板」，在角色扮演过程中同步显示当前世界状态。

### 各部分说明

1. **顶部栏**：显示世界时间、运行状态
2. **左侧场景面板**：当前地点、时间、天气、季节、氛围
3. **角色面板**：玩家的角色信息、在场 NPC 列表
4. **世界观备忘**：修炼体系、势力等关键设定
5. **右侧剧情演绎区**：过往的剧情记录（旁白/对话/事件）
6. **底部状态栏**：当前日期和时间

## 如何在角色扮演中使用

### 方式一：手动更新（推荐）

每次我发出以下格式的信息时，你可以手动更新网页：
```
[面板更新]
场景: 天阑城坊市 | 正午 | 晴 | 热闹喧嚣
角色: 韩立 | 筑基中期
NPC: 黑袍老者(中立) 店小二(友好)
世界: 第 7 日
```

操作步骤：
1. 打开 `index.html`
2. 在浏览器开发者工具（F12）的控制台中输入：

```javascript
// 更新场景
MirrorRealm.setScene({name:'天阑城坊市',time:'正午',weather:'晴',atmosphere:'热闹喧嚣',tags:['坊市','修仙者聚集']});

// 更新玩家
MirrorRealm.setPlayer({name:'韩立',title:'筑基期散修',avatar:'韩',realm:'筑基中期'});

// 更新NPC
MirrorRealm.setNPCs([{name:'黑袍老者',attitude:'neutral',state:'中立'},{name:'店小二',attitude:'friendly',state:'招呼客人'}]);

// 添加剧情条目
MirrorRealm.addEntry('narration', {text:'你走在坊市中...', timeTag:'正午·天阑城'});
MirrorRealm.addEntry('dialogue', {speaker:'黑袍老者', avatar:'老', avatarClass:'npc-neutral', text:'小友，留步。', action:'转身看向你'});
MirrorRealm.addEntry('event', {text:'一道火光划破天际！', severity:'danger', label:'⚡ 突发事件'});

// 设置世界信息
MirrorRealm.setWorld('凡人修仙传', '金鼓原战场篇');
MirrorRealm.setWorldTime('第7日 正午');
MirrorRealm.setStatus('running');
MirrorRealm.setDay(7);

// 暂停提示
MirrorRealm.showPauseHint();

// 清空剧情区
MirrorRealm.clearStory();
```

### 方式二：快捷更新

你也可以简单修改 `index.html` 文件底部的 `<script>` 标签，
把 `MirrorRealm.setWorld(...)` 等初始值写成固定值，
每次在浏览器中刷新即可看到更新。

## 不需要网页面板也能玩

如果觉得更新网页太麻烦，完全可以**只在对话中进行角色扮演**。
网页面板只是一个可选的氛围增强工具，不是必需组件。
